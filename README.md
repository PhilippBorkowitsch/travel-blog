# ts-express-template
Some simply typescript-express templates. 
## List
- Blank app
- Web app
- Api app
- ect.

## Techs
- Typescript
- Express
- Nodemon

## Installation

```sh
# clone repo
git clone https://github.com/jinhduong/ts-express-template.git
# install packages
npm install
# run build and watch typescript files
tsc:s
# run dev mode (nodemon)
npm run dev
# open localhost:3000 in browser
```
